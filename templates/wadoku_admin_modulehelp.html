Welcome to the WaDoku

The goal from this module is to create a dictionry for Japanese words.

Therefore you can input "Kanji", pronunciation in "Hiragana", roman letters "Romaji", a transcription,Translation words.

We recommend to use the "Tag" for your option. If you like to learn or to sort the words.

If you like, you can show a block. The block shows in every page a random word from you.

The module shows in the front end a "Quick search". Take this search to find words in "Kanji", "Hiragana" and in your "Translation".

WaDoku has a global search function for the ImpressCMS too.
